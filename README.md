## About

LibyFlix is a modern streaming platform built using React, TypeScript and Vite, providing a fast and responsive experience for browsing movies and TV shows.

---

## Features

- Responsive UI
- Fast Performance
- Movies & TV Shows
- Search
- Favorites
- Modern Design

---

## Tech Stack

- React
- TypeScript
- Vite
- CSS

---

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Production

```bash
npm run build
npm run preview
```
